## 📺 Getting Started

#### 🔐 Prerequisites: 

- [ ] Npm

```sh
   npm install npm@latest -g
   ```
  
#### 💾 Installation :

- [ ] Clone the repo :

  ```sh
     git clone https://github.com/jstm17/threejs-solar-system
     ```

- [ ] Install the packages

   ```sh
   npm install
   ```

- [ ] run the project

  ```sh
     npm run dev
     ```
